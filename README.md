# hr-analytics-dashboard
HR Analytics Dashboard built using Tableau to analyze employee count, attrition rate, department-wise trends, age distribution, job satisfaction, and education-level insights. This interactive dashboard helps HR teams identify workforce patterns, improve retention strategies, and support data-driven decision-making.
